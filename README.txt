This is a PyRosetta workshop for course 20.420 Principles of Molecular Bioengineering. Open PyRosetta_Tutorial_2023.ipynb in Google Colaboratory. The only non-standard library dependency should be PyRosetta.  

Contents are almost entirely derived from Patrick Holec's version of this PyRosetta workshop (https://github.com/PatrickHolec/PyRosetta_Tutorial_20420), with minor compatibility updates from Rosetta Common (https://rosettacommons.github.io/PyRosetta.notebooks/)
